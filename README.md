## Backend

### Setup

1. Make sure you've already copy pasted `.env.example` to `.env`
2. Make sure you've already run `pnpm install`
3. Run `pnpm set-up api` this will setup the docker, the database, and run the migrations
4. Run `pnpm start:api` to run the api server
